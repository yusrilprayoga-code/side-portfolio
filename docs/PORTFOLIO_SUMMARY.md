# Rangkuman Portofolio (Portfolio Summary)

## Tentang Proyek Ini
Proyek website portofolio ini dimiliki dan dikembangkan oleh **Yusril Prayoga**. 

Ini adalah website portofolio pribadi modern yang dibangun untuk memamerkan proyek, artikel (blog), dan sertifikat secara profesional milik Yusril. Proyek ini sangat berfokus pada performa (kecepatan), SEO, dan pengalaman pengguna (UX) yang interaktif.

## Teknologi Utama yang Digunakan (Tech Stack)
- **Framework:** Next.js (dengan App Router) & React.
- **Bahasa Pemrograman:** TypeScript.
- **Styling:** Tailwind CSS dipadukan dengan Framer Motion untuk animasi mulus (smooth transitions) dan efek 3D (seperti komponen ThreeDCard).
- **Manajemen Konten (Blog):** MDX (Markdown dengan React components).
- **Fitur Khusus:**
  - **Progressive Web App (PWA):** Website ini mendukung akses offline dan bisa diinstal layaknya aplikasi native (di HP maupun desktop).
  - **Live Visitor Counter:** Dilengkapi dengan fitur pelacakan pengunjung secara *real-time* berbasis API.
  - **AI Chatbot (RAG):** Terintegrasi dengan sistem asisten pintar lokal yang dibangun menggunakan Python (TF-IDF, Scikit-learn) dan LLM (seperti GPT-OSS-20B/Gemini) untuk menjawab pertanyaan seputar codebase.

## Struktur & Halaman Utama
1. **Home (Beranda):** Menampilkan perkenalan diri, animasi hero yang menarik, dan cuplikan proyek unggulan.
2. **Projects:** Menampilkan daftar proyek yang pernah dikerjakan (dengan fitur filter/kategori).
3. **Blog:** Tempat membagikan artikel-artikel teknis seputar programming (membaca file dari folder MDX).
4. **Certificates:** Menampilkan galeri sertifikat yang dioptimalkan gambarnya menggunakan format WebP.

## Tujuan Proyek
Proyek ini dibuat tidak hanya sebagai resume online, tetapi juga sebagai tempat *playground* (eksperimen) untuk mengimplementasikan teknologi web terbaru (seperti integrasi AI, animasi kompleks, dan optimasi arsitektur frontend).
